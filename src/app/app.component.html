🌐 About GeoIP

GeoIP is a web application that allows you to localize any IP address on an interactive map and explore detailed information about its geographical location.

Services used
  • ipapi.co — IP geolocation data (city, country, ISP, currency, etc.)
  • ipify.org — detects your current public IP address
  • OpenWeatherMap — current weather for the IP location
  • TimeZoneDB — timezone information and offset
  • Mapbox — interactive map visualization
  • FlagCDN — country flag images
Data presented
  • 📍 IP address, city, country, region
  • 🌐 Latitude / Longitude coordinates
  • 🏢 ISP / ASN information
  • 💱 Currency, population, area, languages
  • 🌤️ Current weather (temperature, wind, pressure, humidity)
  • 🕐 Timezone with live clock
